Class Logger

Abstract class for creating loggers that are configurable, have a source (reference a context), and are capable of logging messages of various log levels.

Configuration parameters

Parameters to pass to the configure method for component configuration:

  • "level" - the LogLevel to set (default is LogLevel.Info);
  • "source" - the logger's source.

Examples

Example usage:

    let logger = new Logger();
    logger.log(LogLevel.Error, "123", null, "error...");
