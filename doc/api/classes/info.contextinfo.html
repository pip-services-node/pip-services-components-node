Class ContextInfo

A simple class that defines the context of execution. Used for various logging functions, where information about the source that is being logging must be known (what's the process's name, what is it, what does it do).

Configuration parameters

Parameters to pass to the configure method for component configuration:

  • "name" - the context's name;
  • "description" - the context's description;
  • "properties.<...>" - additional properties of the context.
