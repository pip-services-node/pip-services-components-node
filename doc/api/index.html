pip-services-components-node


Component definitions for Node.js

This framework is a part of Pip.Services project. It defines standard components types that be used to build applications and services:

  • Auth - authentication credential stores
  • Build - factories
  • Cache - distributed cache
  • Config - configuration readers
  • Connect - connection discovery services
  • Count - performance counters
  • Info - context info
  • Lock - distributed locks
  • Log - logging components
  • Test - test components
